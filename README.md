# Project name- Shopping trends

Analyzed shopping trend- The dashboard makes it evident that the majority of the consumer's purchases are in the "clothing" category with spring season showing the highest sales. Also majority of them use credit card as the payment method. Data also shows most of the men uses promo code and have subscription, whereas Females are not using any promo code. Approximately, 16.9% of them make purchases every three months, with North Dakota having the highest purchase amount.
